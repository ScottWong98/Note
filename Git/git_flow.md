# Git

## Local
``` bash
$ git init
$ git add <filename> / .
$ git commit -m "..."
$ git log
$ git reflog
$ git reset --hard HEAD^
$ git restore --staged
$ git branch <branch>
$ git checkout -b <branch_name>
$ git branch
$ git branch -r
$ git branch -a
```
## Remote
```bash
$ git remote -v
$ git remote show <remote_name>
```
